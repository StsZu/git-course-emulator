::Створення проектної папки::
Яку робочу папку створює автор для всього курсу? {
=my-project (my project)
~temp-course
~git-only
#### На робочому столі створюється папка my-project, яку використовуватимуть протягом усього курсу.
}

::Структура my-project::
Які три субдиректорії створюються всередині my-project? {
=code, data-sources і materials (код, джерела даних, матеріали)
~src, test, deploy
~docs, images, videos
#### Створюються папки для коду, джерел даних і матеріалів — «код», «data sources», «матеріали».
}

::Команда touch::
Для чого використовується touch? {
=Для створення нового файлу з указаною назвою та розширенням
~Для видалення порожніх папок
~Для запуску Python-скрипта
#### touch «потребує лише вказання назви файлу та його розширення» — наприклад, temp-file.txt.
}

::Команда cp для файлів::
Що робить команда cp? {
=Копіює файл у вказане місце, залишаючи оригінал у поточному каталозі
~Переміщує файл без залишення копії
~Показує вміст файлу на екрані
#### cp копіює файл: оригінал залишається, а копія з'являється в цільовій папці, наприклад materials/.
}

::Рекурсивне копіювання::
Яку опцію потрібно додати до cp для копіювання цілої директорії? {
=-r (рекурсивно)
~-a (архівувати)
~-h (допомога)
#### Без опції -r копіювання папки не спрацює: «defice R означає рекурсивно скопіювати» все вміст директорії.
}

Як echo використовується у відео? {

~Для зміни імені користувача Git
#### echo дозволяє «написати щось і додати це в текстовий файл» — наприклад, print('Hello world') у hello-world.txt.
}

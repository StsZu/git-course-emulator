### Імпорт у Moodle

- **`gift/`** — чистий GIFT (лише питання, починаються з `::`) — **рекомендовано для імпорту**
- **`quiz_parts/*.txt`** — повний пакет: транскрипт + quiz

Moodle → Питання → Імпорт → формат **GIFT** → обери файл з `quiz_parts/gift/`.

~Бо він доступний лише на Linux
#### PowerShell «має потужніші можливості для автоматизації», але його синтаксис «може бути бар'єром і складним для новачків».
}

::Мета Git Bash::
Для чого створений Git Bash на Windows? {
=Щоб користувачі Windows могли працювати з інструментами командного рядка Linux
~Щоб замінити Git повністю
~Щоб відкривати лише PDF-файли
#### Мета Git Bash — «забезпечити користувачів Windows працювати з інструментами командного рядка Linux».
}

::Bash на Windows::
Чим технічно є Bash у Git Bash на Windows? {
=Емуляцією, а не повним bash як у Linux
~Точною копією Linux-ядра
~Графічним редактором коду
#### Технічно це «не повний bash як у Linux — це емуляція», але «більш ніж достатньо для більшості завдань».
}

::Командний рядок (Command Line)::
Що таке командний рядок? {
=Текстовий інтерфейс для введення команд операційній системі
~Вікно браузера для перегляду документації
~Папка на робочому столі
#### Command Line — «текстовий інтерфейс для взаємодії з операційною системою», простір для введення команд.
}

::Термінал (Terminal)::
Чим термінал відрізняється від командного рядка? {
=Термінал — це програма/вікно, що надає доступ до командного рядка
~Термінал і командний рядок — це завжди одне й те саме
~Термінал — це лише назва папки в проекті
#### Термінал — «програма або інструмент, який надає доступ до командного рядка»; це вікно, де вводимо команди.
}

::Shell як інтерпретатор::
Що робить Shell? {
=Інтерпретує команди користувача і передає їх операційній системі
~Зберігає лише історію комітів Git
~Відображає лише графічні іконки файлів
#### Shell — «програма, яка виконує команди»: ми вводимо команду, Shell інтерпретує її і повертає результат у термінал.
}

::Bash і Shell::
Як пов'язані Bash і Shell? {
=Bash — це конкретна реалізація (тип) Shell
~Bash — це синонім GUI
~Shell — це скорочення від Bash
#### Bash — «стандартна оболонка» в Linux/macOS; Shell — загальна концепція, а Bash — її конкретна реалізація.
}

::Домашній каталог::
Де зазвичай знаходиться домашній каталог користувача Windows? {
=На диску C у папці Users\username
~У корені диска D без підпапок
~Лише в папці .git
#### Домашній каталог — «диск C, users, username»; там зберігаються особисті файли користувача.
}

# Git & GitHub Trainer

Інтерактивний тренажер команд Git і Git Bash українською — на основі курсу
[Nikita Tymoshenko «Повний курс по Git та GitHub»](https://youtu.be/9CnZihyYjjA).

Один HTML-файл, без API, працює в браузері.

## Швидкий старт

```bash
open index.html
```

Або GitHub Pages: https://stszu.github.io/git-course-emulator/

## Розділи (7 модулів)

1. **Git Bash** — ls, pwd, cd, mkdir, touch, rm, grep
2. **Налаштування Git** — config, init, status
3. **Базові команди** — add, commit, log, diff
4. **Розширення** — amend, diff --cached, restore --staged
5. **Відновлення** — restore, reset, rm, show
6. **Гілки** — branch, checkout, merge
7. **GitHub** — remote, push, pull --rebase, ssh

## Можливості

- Емуляція Git Bash + простий in-memory репозиторій
- Українські підказки (жовтий текст) під кожною командою
- Прогрес по розділах
- **Режим тестування** — питання з підказок у випадковому порядку
- **Обрати сценарій** — перехід між розділами курсу
